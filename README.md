# CodeLab

CodeLab é uma plataforma de estudos e prática de programação com foco em ajudar estudantes do curso de TSI no IFPE no inicio do curso esse projeto esta sendo desenvolvido e acompanhado para a materia de projeto 03.

## Objetivo

O projeto foi desenvolvido com o objetivo de fornecer um ambiente de aprendizagem interativo, permitindo que estudantes pratiquem programação e recebam feedback automatizado sobre suas respostas, para isso utilizamos questoes de semestres anteriores para assim gerar uma maior familiaridade com possiveis proximas questoes de provas.


## Tecnologias Utilizadas

### Backend

* Spring Boot
* NodeTs

### Banco de Dados

* Mysql

### Infraestrutura

* Docker

### FrontEnd

* React

Projeto desenvolvido para fins de estudo e aprendizado.
